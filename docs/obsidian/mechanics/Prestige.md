# Prestige

Prestige is a reset for when a run has reached a milestone. It starts the next
run from a higher level instead of level 1, and turns completed milestones into
rubies and permanent stars.

Use Prestige to unlock stronger options for later runs.

## How It Works

1. Reach a level-10 milestone.
2. Claim the milestone from the Prestige room.
3. Review the next-run preview, then confirm the reset.
4. Start the new run with your milestone rewards and stars.
