# Prestige And Automation

Prestige records completed level milestones and resets the run. Ruby current is
derived from completed milestones at run start. Emerald and personal task
progress persist across prestige resets.

Automation turns advanced research into repeated seed, garden, and cauldron
actions. Automation observes other gameplay facades and runs after normal timers
each ECS tick; the original systems still own their own rules.
