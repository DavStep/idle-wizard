# Garden Plots

A plot is a purchased planting slot. Player level or permanent capacity research raises the **buy cap**; it never grants the plot for free.

![[mechanics/garden/Plot Catalog.base#Plots]]

## Unlock shape

- Plot 1 starts unlocked.
- Plots 2–5 become buyable through player-level caps.
- Plots 6–12 require ordered permanent capacity research and completed Prestiges.
- Every newly available plot must still be purchased with coin.
- All plots share the same crop rules and can grow every seed.
