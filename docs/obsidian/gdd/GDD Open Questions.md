# GDD Open Questions

## Product

- What is the long-term fantasy of prestige beyond earning ruby and opening
  advanced research lanes?
- How social should guilds and trade alliances become before they need deeper
  backend authority?
- Which world events should become recurring event families, and which should
  stay one-off world news?

## UX

- How much guidance should remain after the first session once the player knows
  the core loops?
- Which room should explain advanced capacity research: Research, Prestige, or
  the room that benefits from the capacity?

## Balance

- How visible should decade walls be before the player reaches them?
- Which late-game resource should carry the most pressure: coin, time, ruby, or
  emerald?
