# Tutorial Surface Rules

## Single Main Surface

Keep one main guide surface visible at a time. Do not stack a target hint box
with an objective box. Pointer cues may coexist with the objective panel only
when the objective asks for target guidance.

## Elara Surfaces

- The first purchase dialog is a free workshop-entry dialog.
- A small Elara button stays on the left side.
- Pressing it opens the current lesson panel.
- Press-to-advance steps use the same panel with border actions like `next` or
  `continue`.
- Major first-time concepts get short checkpoint prompts after completion.

## Style

Tutorial boxes follow the project sparse text-game style: compact source text,
embedded border labels, no rounded cards, no gradients, no decorative target
frames, and pointer-only target cues.
