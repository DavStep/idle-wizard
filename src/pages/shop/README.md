# Shop Page

The Shop is the room to the right of Workshop.

It shows one `shop shelf` block. The shelf displays up to five slots. Slot 1 starts unlocked, locked slots show their gold cost, and only the next locked slot can be bought. Selected slots show the current sell price from the gameplay snapshot. Selecting an unlocked slot opens a popup where the player chooses an exact item to auto-sell, grouped under `seeds`, `herbs`, and `potions` tabs, with live sell prices beside each item.
