# Top Panel

The top panel is shared room UI. It shows player name, mana, gold, and crystal above each page.

Use two compact rows: the username gets the full first row and opens settings. Settings contain username editing plus visual theme, font, and color selectors. Visual option names select researched options; zero-cost unresearched names unlock and select in one tap. The right-side price/status label researches locked paid options. Mana, gold, and crystal share the second row. Shrink that row's source font only when resource text would clip. Keep it plain with the ordinary `1px` panel treatment.
