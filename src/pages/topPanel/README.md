# Top Panel

The top panel is shared room UI. It shows player name, mana, gold, and crystal above each page.

Use two compact rows: the username gets the full first row and opens settings. Settings contain username editing plus visual theme, font, and color selectors. Visual option names select researched options only. The right-side price/status label, including `free`, researches unresearched options; after that the name can select the option. Mana, gold, and crystal share the second row. Shrink that row's source font only when resource text would clip. Keep it plain with the ordinary `1px` panel treatment.
