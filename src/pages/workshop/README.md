# Workshop Page

The Workshop is the first room view.

It shows the wall and floor of the place where later crafting interactions will live, the `mana sphere` block, the summon button outside that block, the seed inventory popup opened from the `seeds` row, and the page name at the bottom center. Follow `docs/style.md`: black text, white surfaces, readable serif text with some character, simple non-dialog boxes, and no decorative visuals. Herb, potion, and recipe item concepts exist in gameplay items; planting, active brewing, selling, and economy logic should be added through dedicated gameplay facades later.
