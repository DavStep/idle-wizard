# Workshop Page

The Workshop is the first room view.

It sits between Garden and Research. It shows the `level N requirements` box, summon action, bag popup, prestige action, transient reward flyouts, logs, discoveries, leaderboard, alliance controls, and page name. Mana amount and regen live in the shared top panel. Workshop secondary actions unlock by player level; keep those gates in page managers and keep gameplay effects behind gameplay/backend facades.

Follow `docs/style.md`: black text, white surfaces, readable Lexend text, simple non-dialog boxes, and no decorative visuals. Reuse shared row, popup, border-label, and expandable-box patterns from `docs/ui-patterns.md`.
