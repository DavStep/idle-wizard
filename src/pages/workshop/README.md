# Workshop Page

The Workshop is the first room view.

It sits between Garden and Research. It shows the wall and floor of the place where later crafting interactions will live, the `mana sphere` block, a separate `seeds` block with the summon seed action, transient summon result flyouts, the seed inventory popup opened from the `seeds` row, a leaderboard popup with single-player/alliance target tabs and daily/weekly/monthly/all-time period tabs, and the page name at the bottom center. Follow `docs/style.md`: black text, white surfaces, readable Lexend text, simple non-dialog boxes, and no decorative visuals. Herb, potion, and recipe item concepts exist in gameplay items; planting, active brewing, selling, and economy logic should be added through dedicated gameplay facades later.
