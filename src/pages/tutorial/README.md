# Tutorial

The tutorial points at real room controls for the first loop: tasks, sage seed research, seed summon, sage growing, then one task finish loop.

It does not grant items, unlock rooms, change gameplay rules, or hide systems. Skip only stores that the guide overlay should stay hidden.

Players already past level 1 auto-complete the tutorial. Level 1 snapshots that already show later progress skip earlier lessons instead of replaying stale guidance.
