# Brewing Page

Brewing is the left edge room page, immediately left of Garden.

It renders a pannable brewing world, cauldron nodes, bottom icon buttons that expand inline herb and potion boxes, unlocked cauldron contents in the cauldron dialog, the next buyable cauldron, brew and bottle actions, per-cauldron recipe selection and auto state inside the cauldron dialog, navigation, and the bottom page name.
It does not own potion recipes, brewing costs, brew timers, inventory changes, effects, or progression rules; those stay in `src/gameplay/brewing/`.
