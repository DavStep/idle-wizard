# Visual Settings

Visual settings prices turn profile-only choices into researchable options. White, Lexend, regular progress bars, and the default avatar set (`elara`, `mira`, `bramble`, `corvin`, `juniper`, `rowan`) start researched. Resource colors and icons are fixed on; stale `monochrome` or `none` values normalize back to `resources` and `icons`. Unresearched choices, including zero-cost `free` choices, must be researched from the price label before their name can select the option. The avatar picker is the exception: locked avatars show as silhouettes until another flow marks them researched.
