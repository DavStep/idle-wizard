# Prestige

Prestige records completed level milestones and resets the run. Completed milestones are permanent prestige data. Ruby current is derived from those completed milestones at run start, not treated as permanent currency by itself.

Milestones are every 10 levels. The default reward is `1 ruby`, with `level 50 = 2 ruby` and `level 100 = 5 ruby`.
